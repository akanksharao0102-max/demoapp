# Demo Laravel Web Application

## Project Description
This is a basic web application developed using Laravel.  
It includes multiple pages like Home, About, Services, and Contact with a contact form.

## Features
- Navigation bar
- Multiple scrollable sections
- Contact form with validation
- Data stored in database
- Custom CSS styling

## Technologies Used
- Laravel
- PHP
- HTML
- CSS
- MySQL

## Installation Steps

1. Clone the repository:
   git clone https://github.com/YOUR-USERNAME/demoapp.git

2. Go inside project folder:
   cd demoapp

3. Install dependencies:
   composer install

4. Start server:
   php artisan serve

5. Open in browser:
   http://localhost:8000
